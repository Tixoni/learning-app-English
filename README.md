📄 Формат файла вопросов (questions.txt)
# Название темы

Текст вопроса | Правильный ответ | Неверный вариант 2 | Неверный вариант 3 | Неверный вариант 4
Правила:

# в начале строки = заголовок темы
Строки с | = вопрос + варианты ответов
Первый вариант после | = правильный ответ
Остальные варианты (3–4 штуки) = неверные варианты для режима «Выбор»
Если вариантов меньше 4 — приложение добоёрет ответы из других вопросов
Строки // = комментарии
Пустые строки игнорируются
